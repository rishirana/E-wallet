1. This is the Android Wallet App like paytm and mobikbik etc.

2. You can copy the .apk file from the apk folder and paste it 
   in the your android mobile sd card and then run the application.

3. The application is now installed on your android mobile.

4. This application looks like the screenshot given in the screenshot 
   folder.

5. You can also see the source code and all the library used in the
   application by going into the Bank2 folder and then inside app folder.

6. The Database name is bankproject and there are five table inside bankproject 
   database  bank, bankaccount , editaccount, minitransaction, userapplication.
